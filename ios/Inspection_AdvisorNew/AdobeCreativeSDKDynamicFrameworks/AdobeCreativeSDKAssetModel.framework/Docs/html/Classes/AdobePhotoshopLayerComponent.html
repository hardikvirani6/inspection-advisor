AdobePhotoshopLayerComponent Class Reference

Inherits from NSObject
Declared in AdobePhotoshopLayer.h

Overview

Represents a layer component. A component describes the metadata necessary for an “external” blob of data. This external data could be a pixel layer’s actual bitmap information, a pixel mask’s bitmap data, an embedded smart object or other relevant information.

Overview

AdobeAssetPDFFile represents an PDF Document. It provides methods for creating PDF files.

Parameters

name

The name of the file to create.

folder

The name of the AdobeAssetFolder containing the PDF file to be created.

The AGC manifest used to create the PDF file on Adobe Creative Cloud.

imageComponents

An array of AdobeAGCImageComponent objects.

overwrite

Whether to allow overwriting an existing destination file.

progressBlock

Optionally, track the upload progress.

successBlock

Optionally, get an updated reference to the created file when complete.

cancellationBlock

Optionally, be notified of a cancellation on upload.

errorBlock

Parameters

name

The name of the file to create.

folder

The name of the AdobeAssetFolder containing the PDF file to be created.

images

Array of AdobeAssetFile objects for the creation of the AdobeAssetPDFFile. These objects have to be SVG files without external references that may contain base64 encoded JPEG or PNG images. (Currently only the first array object is used).

overwrite

Whether to allow overwriting an existing destination file. Set to NO if the file doesn’t exist.

progressBlock

Optionally, track the upload progress.

successBlock

Optionally, get an updated reference to the created file when complete.

cancellationBlock

Optionally, be notified of a cancellation on upload.

errorBlock

Optionally, be notified of an error.

Return Value

A place holder pointer to the AdobeAssetPDFFile.

Discussion

Create an PDF file asynchornously.

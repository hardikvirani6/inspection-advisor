imagesByNumberOfPixelsInRange

Search for images that contains the number of pixels in a specified range Pixel size is defined as multiplication of height and width.

@property (assign, nonatomic) NSRange imagesByNumberOfPixelsInRange

Discussion

Search for images that contains the number of pixels in a specified range Pixel size is defined as multiplication of height and width.

Example: If height is 1,920 pixel and width is 1,080 pixel, then pixel size is 2,073,600 ≈ 2 mega pixels.

Overview

assetsMatchingQuery:offset:pageSize:requestPriority:completion:

Fetch the assets by its query.

+ (nonnull AdobeNetworkHTTPRequest *)assetsMatchingQuery:(nonnull AdobeUniversalSearchQuery *)query offset:(NSInteger)offset pageSize:(NSInteger)pageSize requestPriority:(NSOperationQueuePriority)requestPriority completion:(void ( ^ _Nullable ) ( NSArray<AdobeUniversalSearchAsset*> *_Nullable assets , NSInteger count , NSError *_Nullable error ))completion

Parameters

query

The query object specifying the search to perform.

offset

The offset of the returned data from within the entire search set. Use this to page over the search set.

pageSize

The maximum number of media items to return in this search. Allowed values: 1 - 64

requestPriority

The relative priority of the HTTP request.

completion

Called when the method has finished.

Return Value

An AdobeNetworkHTTPRequest object that can control the priority and allow for cancellation of the request.

Discussion

Fetch the assets by its query.
